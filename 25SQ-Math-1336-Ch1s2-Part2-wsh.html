Integral Calculus with Parametric Curves:

Consider a curve C on the interval a ≤ x ≤ b with parametric equations

x = x(t),y = y(t),α ≤ t ≤ β

We would like to extend the concepts of area and arclength from Integral Calculus to the new setting of parametric curves.


Figure 1: Diagram illustrating ﬁnding the area under a parametric curve.

 

Area:
To calculate the area under the curve y = F(x) from x = a to x = b:

A = ∫ abF(x)dx = ∫ abydx = ∫ αβy(t)x′(t)dt

The set-up above is for a curve that is parametrized from left to right:

a = x(α),b = x(β).

If the curve is parametrized from right to left, the limits of integration should be swapped so that the lower limit of integration corresponds to the left endpoint of the interval, x = a, and the upper limit of integration corresponds to the right endpoint of the interval, x = b.


Figure 2: Diagram illustrating ﬁnding the length of a parametric curve.

 

Arclength:
To calculate the length, L, of the parametric curve from t = α to t = β, set up the following integral, where ds is the inﬁnitesimal distance element along the curve:

L = ∫ ds = ∫ αβdx dt 2 + dy dt 2dt

Note that the expression for ds comes from the Pythagorean theorem, and the idea that in the limit as each of the distance elements go to zero:

ds dt2 = dx dt 2 + dy dt 2
