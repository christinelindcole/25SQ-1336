Alternating Series Test (AST):_____________________________________________________

 
If an alternating series can be written in one of the following forms:
 
∑ n=1∞(−1)n+1b n = b1 − b2 + b3 − b4 + b5 − b6 + … = ∑ n=1∞(−1)n−1b n
 
 
-OR-
 
∑ n=1∞(−1)nb n = −b1 + b2 − b3 + b4 − b5 + b6 −…
 
 
and satisﬁes both of the conditions listed below, then the series is convergent.

(i)
0 < bn+1 ≤ bn
(ii)
limn→∞bn = 0

Alternating Series Estimation/Remainder Theorem:_________________________

 
If S is the sum of a convergent alternating series

∑ n=1∞(−1)n+1b n-OR-∑ n=1∞(−1)nb n,

then

|RN| = |S − SN|≤ bN+1

Absolute & Conditional Convergence: __________________________________________

A series ∑ an is called absolutely convergent if the series ∑ |an| is convergent.
 

A series ∑ an is called conditionally convergent if it is convergent, but not absolutely convergent.

Theorem 5.15:
If a series ∑ |an| converges, then ∑ an converges.
