Taylor/Maclaurin Series Summary:

Key Idea: When x is close to a and n is large: the nth degree Taylor/Maclaurin polynomial should approximate f(x) very well!
_________________________________________________________________________________________

Definition for Taylor Series for f(x) centered at a:

∑ n=0∞f(n)(a) n! (x − a)n = f(a) + f(1)(a)(x − a) + f(2)(a) 2! (x − a)2

+f(3)(a) 3! (x − a)3 + f(4)(a) 4! (x − a)4 + …

Definition for Maclaurin Series for f(x):

∑ n=0∞f(n)(0) n! xn = f(0) + f(1)(0)x + f(2)(0) 2! x2 + f(3)(0) 3! x3 + f(4)(0) 4! x4 + …

Taylor’s Formula / Lagrange’s Form of the Remainder:

Rn(x) = f(n+1)(z) (n + 1)! (x − a)n+1,

Alternating Series Estimation Theorem Formula:

|Rn| = S − Sn ≤ bn+1

Selected Maclaurin Series with Radius of Convergence, R:

ex = ∑ n=0∞xn n! = 1 + x + x2 2! + x3 3! + x4 4! + …,R = ∞

sin(x) = ∑ n=0∞(−1)nx2n+1 (2n + 1)! = x −x3 3! + x5 5! −x7 7! + x9 9! −…,R = ∞

cos(x) = ∑ n=0∞(−1)nx2n (2n)! = 1 −x2 2! + x4 4! −x6 6! + x8 8! −…,R = ∞

Geometric Series:  1 1 − x = ∑ n=0∞xn = 1 + x + x2 + x3 + x4 + …,R = 1

ln(1 + x) = ∑ n=1∞(−1)n−1xn n = x −x2 2 + x3 3 −x4 4 + x5 5 −…,R = 1
